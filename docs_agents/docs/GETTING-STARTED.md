# 시작하기 — 비개발자도 따라 하는 사용 설명서

이 도구는 **AI 문서 작성 팀**입니다. 기획서·요구사항·회의록 같은 문서를, 형식에 맞게
대신 써주고 검토해주고 Google Drive에 정리까지 해줍니다. 개발 지식이 없어도 됩니다.

> 한 줄 요약: **Claude Code를 켜고 → 한국어로 시키면 → AI가 문서를 만들어 Drive에 올려줍니다.**

---

## 이게 뭔가요? (1분 개념)

- **여러분이 하는 일**: "이런 문서 만들어줘"라고 **말로 시키기**.
- **AI가 하는 일**: 담당 전문가 AI가 표준 양식에 맞춰 **초안 작성 → 검토 → Drive 저장**.
- **문서가 저장되는 곳**: 팀이 함께 보는 **Google Drive**. (이 폴더가 아니라 Drive 웹에서 봅니다)

문서 종류마다 담당 AI가 따로 있습니다 (기획자·요구사항 분석가·UX 설계자·QA 등).
여러분은 누가 담당인지 몰라도 됩니다 — 그냥 시키면 알아서 맞는 담당이 나섭니다.

---

## 준비 (최초 한 번, 보통 IT 담당이 도와줌)

처음 한 번만 설정하면 그다음부턴 바로 쓸 수 있습니다.

1. **Claude Code 설치** — [claude.com/claude-code](https://claude.com/claude-code)
2. **이 폴더 받기** — 팀에서 받은 주소로 내려받기 (git을 모르면 IT 담당에게 "이 저장소 클론해 주세요"라고 요청).
3. **Google Drive 연결** — 본인 Google 계정으로 1회 로그인.
   → 자세한 그림 설명: **[Google Drive 연결 가이드](GOOGLE-DRIVE-MCP.md)**

> 3번 연결을 안 해도 문서 초안은 받을 수 있어요. 다만 "Drive에 자동 저장"은 연결해야 됩니다.
> (연결 전에는 AI가 만든 글을 복사해서 직접 Drive에 붙여넣으면 됩니다.)

---

## 매일 쓰는 법 (3단계)

### 1단계 — Claude Code 켜기
이 폴더 안에서 Claude Code를 엽니다. (바탕화면 앱이면 이 폴더를 열고, 터미널이면 폴더에서 `claude` 입력)

### 2단계 — 시키기
입력창에 **그냥 한국어로** 쓰면 됩니다. 두 가지 방법이 있어요:

**방법 A — 말로 시키기 (가장 쉬움)**
```
출석 체크 기능 기획서 만들어줘
```
```
이 회의 내용 정리해서 회의록으로 만들어줘
(아래에 회의 메모 붙여넣기)
```

**방법 B — 빗금(/) 명령 쓰기**
입력창에 `/` 를 치면 만들 수 있는 문서 목록이 뜹니다. 골라서 쓰면 됩니다.
```
/planning-doc 출석 체크 기능
```

> 정보가 부족하면 AI가 **되물어봅니다.** ("타깃 사용자가 누구인가요?" 등) 편하게 답하면 됩니다.

### 3단계 — 검토하고 저장하기
초안이 나오면 이렇게 이어서 시키면 됩니다.
```
검수해줘            ← 잘못된 곳·빠진 곳을 짚어줍니다
1, 3번 의견대로 고쳐줘   ← 원하는 것만 골라 반영
드라이브에 올려줘        ← Google Drive에 보기 좋은 서식으로 저장
```

끝나면 **문서 링크**를 알려줍니다. 그 링크를 카톡·메일로 공유하면 됩니다.

## 자주 묻는 질문 (FAQ)

**Q. 빗금(/) 명령을 꼭 외워야 하나요?**
아니요. 그냥 "회의록 만들어줘"처럼 한국어로 말해도 알아서 맞는 기능이 동작합니다.

**Q. 만든 문서는 어디서 보나요?**
Google Drive(웹)에서 봅니다. 저장이 끝나면 AI가 링크를 줍니다.

**Q. 기존 문서를 고치고 싶어요.**
"드라이브에 있는 ○○ 문서 읽어서 △△ 부분 고쳐줘"라고 하면 됩니다. (덮어쓰기 전에 확인을 받습니다)

**Q. AI가 엉뚱하게 길게 분석해요. 토큰이 아까워요.**
기본은 간결 모드입니다. 깊은 다각도 분석이 필요할 때만 "철저히/다각도로 분석해줘"라고 하세요.

**Q. 큰 의사결정 문서는 더 꼼꼼히 봐줬으면 좋겠어요.**
그럴 땐 AI가 먼저 "이건 깊게 분석할까요?"라고 물어봅니다. "응"하면 여러 관점으로 검증합니다.

| 증상 | 해결 |

여전히 막히면 팀 IT 담당이나 이 도구를 세팅해 준 사람에게 위 증상을 그대로 전해 주세요.
