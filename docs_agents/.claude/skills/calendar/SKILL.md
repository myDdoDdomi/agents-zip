# /calendar — 일정 조회·생성·수정

`calendar-manager` 에이전트를 사용해 **Google Calendar** 일정을 다룹니다.

## 절차
1. 요청 유형을 구분한다: **조회/요약** vs **생성/수정** vs **삭제**.
2. `calendar-manager`에게 위임한다.
   - 조회: 기간·캘린더를 받아 표로 정리.
   - 생성/수정: 대상·일시·참석자·캘린더를 **확인한 뒤** 실행(추측 금지).
   - 삭제·대량 변경: 무엇을 지우는지 보여주고 **명시적 승인** 후 실행.
3. 회의 일정이면 `/meeting-notes`, 릴리즈 일정이면 `/release-notes` 연계를 안내한다.

> 캘린더 조회는 자유, 변경(생성/수정/삭제)은 사람 확인을 거칩니다. (Human-in-the-loop)
