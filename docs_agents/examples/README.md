# examples — 서식판 예시 모음

이 폴더는 **다른 팀이 바로 참고할 수 있는 범용 예시**입니다. 특정 프로젝트(고객·서비스명)에
묶이지 않은 **가상의 샘플**로 작성돼 있습니다.

- 실제 산출물은 Google Drive에 **네이티브 서식(제목 스타일·표·브랜드색)** 으로 만들어집니다.
  여기 마크다운 예시는 그 **구조와 구성 요소**를 보여주기 위한 것입니다.
- 서식 규칙의 원본 기준: [`../docs/DOC-FORMATTING.md`](../docs/DOC-FORMATTING.md)
- 실제로 Drive에 올릴 때는 `/drive-sync` → `doc-formatter`가 이 구조를 Google Docs/Sheets
  네이티브 서식으로 변환합니다.

## 목록
| 파일 | 무엇의 예시인가 |
|---|---|
| `예시_분석리포트_서식판.md` | 검수/분석/프리모템류 보고서의 표준 레이아웃(커버 헤더·메타표·위험표·강조박스) |

> 새 팀이 시작할 때: 이 예시의 **구조만 복사**하고 내용은 자기 프로젝트로 바꾸면 됩니다.
> 프로젝트명·고객명 같은 식별 정보는 예시에 넣지 마세요(공유용 중립 유지).
