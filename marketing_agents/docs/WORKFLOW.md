# 작업 워크플로우

마케팅 작업은 5단계 루프를 따릅니다. 그룹장(`CLAUDE.md`)이 라우팅·종합·게이트를 맡습니다.

```
① 리서치(Research) → /market-research        (market-researcher)
② 전략(Strategy)   → /campaign-brief, /positioning (marketing-lead)
③ 생성(Create)     → /blog-post /landing-copy /email-sequence /seo-brief /social-calendar (병렬)
④ 검수(Review)     → /copy-review            (brand-voice-reviewer)
⑤ 핸드오프(Handoff)→ design(비주얼)/dev(기술SEO)/docs(사양) + (옵션) 발행
```

## 협업 흐름

```
요청
  │
  ▼
[marketing-lead] ── 전략·캠페인 브리프·포지셔닝
  │
  ├──▶ [market-researcher] ── 근거(시장·경쟁·오디언스)
  │
  ├──▶ [content-writer] ── 블로그·롱폼
  ├──▶ [copywriter] ── 랜딩·광고·이메일 (+EN 옵션)
  ├──▶ [seo-specialist] ── SEO 브리프·온페이지
  ├──▶ [social-media-strategist] ── 소셜·숏폼
  │
  ▼
[brand-voice-reviewer] ── 보이스·사실·규정 검수 게이트
  │
  ▼
[marketing-lead] 종합 ──▶ design(비주얼) / dev(기술SEO) / docs(사양) 핸드오프
```

## 깊이 모드 (효율)
- **빠르게**: 단건 카피·소셜 포스트 → 담당 1명.
- **표준(기본)**: 일반 콘텐츠·SEO·이메일 → 담당 1~2명 + 그룹장 요약.
- **심화**: 신규 캠페인·포지셔닝 재정립 → 다관점 병렬 + 교차검증(그룹장 제안·승인).

## 게이트(사람 승인) 항목
- 소셜/이메일 **외부 발행·예약**, 유료 광고 집행, 미검증 주장 포함 콘텐츠.
