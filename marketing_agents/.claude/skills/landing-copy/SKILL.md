# /landing-copy — 랜딩·광고 카피

`copywriter` 에이전트를 사용해 **랜딩/광고 카피**를 만듭니다.

## 절차
1. `templates/랜딩카피.md` 구조를 확인한다.
2. (있으면) 포지셔닝·캠페인 브리프를 참조한다. 타깃·오퍼·핵심 혜택을 확인한다.
3. `copywriter`에게 위임해 섹션별(히어로·가치·증거·반론해소·FAQ·CTA) 카피와 헤드라인 변형 2~3개를 작성한다.
4. 채널 글자수(`docs/CHANNEL-GUIDE.md`) 준수. 요청 시 **영문(EN) 버전** 병행. 완료 후 `/copy-review` 안내.

인자(`$ARGUMENTS`)가 있으면 대상으로 사용하고, 없으면 타깃·오퍼를 먼저 묻는다.
