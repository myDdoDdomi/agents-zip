# /positioning — 포지셔닝·메시징

`marketing-lead` 에이전트를 사용해 **포지셔닝·메시징**을 정립합니다.

## 절차
1. `templates/포지셔닝메시징.md` 구조를 확인한다.
2. `market-researcher`의 경쟁·오디언스 근거를 참조한다(없으면 `/market-research` 먼저 권장).
3. `marketing-lead`에게 위임해 타깃·경쟁 프레임·차별점·가치 제안·메시지 하우스(핵심 메시지+증거)·톤을 정리한다.
4. 완료 후 이 포지셔닝을 콘텐츠·카피·소셜의 기준으로 삼게 안내한다.

인자(`$ARGUMENTS`)가 있으면 대상으로 사용하고, 없으면 먼저 묻는다.
