# /email-sequence — 이메일 시퀀스

`copywriter` 에이전트를 사용해 **이메일 시퀀스**를 만듭니다.

## 절차
1. `templates/이메일시퀀스.md` 구조를 확인한다.
2. 시퀀스 목적(온보딩/너처/재참여/프로모), 대상 세그먼트, 트리거를 확인한다.
3. `copywriter`에게 위임해 단계별(목적·트리거·제목·프리헤더·본문·CTA)로 작성한다. 제목은 변형 2~3개.
4. 발송 시점·간격을 제안한다. 요청 시 **영문(EN) 버전** 병행. 완료 후 `/copy-review` 안내.
   (실제 발송·자동화 연동은 사람 승인/외부 도구 영역)

인자(`$ARGUMENTS`)가 있으면 목적으로 사용하고, 없으면 먼저 묻는다.
